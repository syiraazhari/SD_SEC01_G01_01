INSTALLATION MANUAL

1. Download xampp server through this link https://sourceforge.net/projects/xampp/files/XAMPP%20Windows/7.1.30/xampp-portable-windows-x64-7.1.30-5-VC14-installer.exe/download
2. Download composer installer https://getcomposer.org/Composer-Setup.exe
3. Install all this set ups
4. Copy the extracted folder containing project files in the htdocs file in xampp
5. From project folder open cmd command locate the project folder which is 'cd'
6. Run this command 'php artisan serve' in command prompt
